# V-bind

```js
<template>
  <div>

    <!-- 绑定样式, 如果 isActive 为 True 则 red 样式生效  -->
    <p><a class="bg-black" v-bind:class="{ red: isActive }">犬小哈教程</a></p>

  </div>
</template>

<script setup>
  import { ref } from 'vue'
  // 定义一个响应式变量
  const isActive = ref(true)

</script>

<style>
  .red {
    color: red;
  }
  
  .bg-black {
    background-color: black;
  }
</style>

```
