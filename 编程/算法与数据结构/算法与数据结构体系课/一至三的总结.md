# 阶段一至阶段三的总结

## 线性

- 动态数据；
- 先进先出，队列；
- 先进后出，栈；
- 链表。

## 非线性，树

1. 链表 → 二叉树；
2. 前驱与后继节点，二分搜索树中节点的删除；
3. 元素的上浮与下沉，堆、优先队列；
4. 二叉树的旋转，AVL 树与红黑树。

## 应用场景

- 区间查询，线段树、SQRT 分解；
- 处理字符串，Trie；
- 连接问题，并查集；

## 编程思想

空间换时间，哈希表。
